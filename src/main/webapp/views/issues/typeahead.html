problem with IE8

When using IE8, with the same source code, you will get the following:

The dropdown box is missing. It seems the typeahead is not returning any filtered items after you put something in the textbox.

Solution

IE8 does not support a string "trim" function. Put the following code in your script, and it will work out.
if (!('trim' in String.prototype)) {
String.prototype.trim= function() {
return this.replace(/^\s+/, '').replace(/\s+$/, '');
};
}
